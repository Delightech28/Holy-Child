

# Holy Child School Website

## Pages & Structure

### 1. Landing Page (Home)
- **Hero Section**: Full-width banner with school name "Holy Child School", tagline, and a CTA button ("Enroll Now")
- **About Section**: Brief school history, mission, and vision
- **Programs Section**: Cards showing Primary and Secondary programs with descriptions
- **School Environment**: Photo gallery grid showcasing the school facilities (using placeholder images)
- **Why Choose Us**: Feature cards highlighting key strengths (qualified teachers, modern facilities, extracurriculars, etc.)
- **Testimonials**: Student/parent testimonial cards
- **Footer**: Contact info, social media links, quick navigation links

### 2. Enrollment Page
- Multi-section enrollment form with fields for:
  - Student info (name, date of birth, gender, grade level applying for)
  - Parent/guardian info (name, phone, email, address)
  - Previous school info
  - Program selection (Primary / Secondary)
- Form validation using zod
- Success message on submission (no backend, just a confirmation toast)

### 3. About Page
- Extended school history and philosophy
- Mission, vision, and core values
- Staff/leadership section

### 4. Contact Page
- Contact form, school address, phone, email
- Embedded map placeholder

### Navigation
- Sticky header with school logo/name, nav links (Home, About, Enrollment, Contact)
- Mobile-responsive hamburger menu

### Design
- Clean, professional look with a warm color scheme (blue and gold/yellow accents)
- Modern typography, rounded cards, smooth scroll animations
- Fully responsive for mobile and desktop
